ถึงปีนี้ไม่มีเสื้อฟรีแต่ยังไงก็รู้สึกว่าการไปช่วย contribute project ต่างๆก็ทำให้เราได้เรียนรู้อะไรมากขึ้นนิดๆ (โดยมีของเล็กๆน้อยๆเป็นผลพลอยได้) ก็เลยเจียดเวลาอันน้อยนิดของตัวเองมาเปิด pull request เอาสนุกมั่งเนอะ

### Contributions (อิงจากที่ "Hacktoberfest accepted")

**[Thailand webring](https://github.com/wonderfulsoftware/webring/pull/182)**
อันนี้ไม่มีอะไรมาก พอดีได้ domain ใหม่กับ setup เว็บตัวเองใหม่ให้พ่น RSS Feed ด้วยเลยปรับเพิ่มสักหน่อย

**[WeVis Parliament Watch](https://github.com/wevisdemo/parliament-watch)**
เป็น project ที่จะทำตัวเว็บไซต์ visualize ข้อมูลการเมืองในไทย เช่น การประชุมสภา การโหวตกฏหมายต่างๆ โดยช่วงเดือน hacktoberfest WeVis มีแจก sticker ให้ด้วยถ้าเปิด pull request ให้ตัว project **มีหรือเราจะพลาด**

จริงๆเคย contribute ให้กับ WeVis ตั้งแต่ [they-work-for-us](https://github.com/wevisdemo/they-work-for-us) แล้วก็เลยพอรู้ flow การทำงานคร่าวๆ (ไปรับ issue ที่พี่เค้าเปิดไว้ อ่านดีเทลดีๆ เขียนโค้ด แล้วก็เปิด pull request ให้รีวิวก่อน merge) บวกกับตัว project ใช้ svelte ด้วย เลยเปิดไปหลาย pull request อยู่เหมือนกัน อันนี้เป็น pull request ที่เปิดไป

- [Proposer component](https://github.com/wevisdemo/parliament-watch/pull/31)
- [Footer component](https://github.com/wevisdemo/parliament-watch/pull/15)
- [Legistration process page -- overview section](https://github.com/wevisdemo/parliament-watch/pull/51)

ส่วนใหญ่เน้นทำตัว component ต่างๆกับลองทำตัวหน้าไป คิดว่าน่าจะเป็น project นึงที่ถ้ามีอะไรเปิดๆมาน่าจะไปดึงมาทำอยู่เหมือนกันครับ

**[Bangkok Open Source Festival '23](https://github.com/creatorsgarten/bosf23/pull/8)**

เป็นของมิตรสหาย [creatorsgarten](https://creatorsgarten.org/) ครับ ใครช่วยเปิด pull request ให้กับ repository ของคนไทยด้วยกันตั้งแต่ 3 pull requests ขึ้นไปก็สามารถเปิดเคลมเสื้อได้ ก็ Hacktoberfest เค้าไม่แจกเสื้อ มารับตรงนี้แทนและกันครับ

นั่นแหละครับผลประกอบการของปีนี้ หวังว่าปีหน้าจะมีอะไรสนุกๆให้ช่วย contribute นะครับ
